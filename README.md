El programa permite el ingreso de datos a distintas estructuras trabajadas en clase, por ejemplo a pilas, colas, listas simples, listas enlazadas
y tambien involucra la creacion de BST a base de los datos ingresados y mostrando los tres tipos de orden en los que se pueden colocar los datos del arbol

Las estrucuras utilizadas son colas, pilas, listas simples, listas doblemente enlazdas y arboles binarios

La logica del programa consiste en la clasificacion o asignacion de los datos ingresados por el usuario segun las condiciones definidas en el programa y la creacion de BST unicamente si el usuario lo desea
o lo solicita mendiante la opcion del menu

Para la correcta ejecucion y uso del programa se despliega un menu de opciones donde indica las acciones que se pueden realizar, se debe ingresar el numero de la opcion que se desea y el programa solicitara
o mostrara lo definido en la opcion seleccionada, si se ingresan datos invalidos el programa lanzara un mensaje indicando la invalidez de lo ingresado y en algunos casos dira lo que se debe hacer para usar 
la funcion
